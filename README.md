# AutoHotkey 脚本与学习笔记仓库

这个仓库用于保存我在使用 [AutoHotkey](https://www.autohotkey.com/) 过程中的脚本代码和学习笔记。内容将包括：

- 🚀 常用的自动化脚本
- 🧠 学习 AutoHotkey 的语法笔记
- 🔧 实用的快捷键替换、窗口管理等示例
- 🗂️ 按功能分类整理的 AHK 文件

## 📁 仓库结构（规划中）

```
/scripts      # 存放所有 AHK 脚本文件
/notes        # 学习笔记和技巧总结
README.md     # 项目说明文件
```

## 📌 使用说明

你可以将 `.ahk` 文件下载到本地后双击运行，或者右键选择 “使用 AutoHotkey 运行”。

建议使用 [AutoHotkey v1.1](https://www.autohotkey.com/) 或 [v2.0](https://www.autohotkey.com/v2/)（请注意版本差异）。
